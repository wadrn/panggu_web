配置数据

配置数据包括rpn部分的配置和cnn后验证部分的配置。

1. 样本类型配置

1）主流程的“配置数据”阶段，点击“点击添加”，出现属性树，选择相应属性，分别对“地平线开放数据”和“我的数据进行配置；
2）点击“确定”，保存选择的属性，并进行RPN样本配置

2. RPN样本配置

a)正样本配置

参数名称 参数解释和范围
感受野宽度 默认值40，有效范围 [24, 72]
感受野高度 默认值40，有效范围 [24, 64]
样本图像宽度 默认384，有效范围[128, 512]
样本图像高度 默认256，有效范围[128, 512]
标准长度 物体在样本图像中的标准长度。默认值32.0，有效范围 [16, 72]
规范标准长度的方法 规范化物体在样本图像中大小的方法，包括根据高度，宽度，对角线，面积的平方根，周长的四分之一几种方法

b)正样本扰动配置

参数名称 参数解释和范围
坐标回归区域是否为椭圆 若检查目标长宽比大于2，则选True
样本最大旋转角度 样本图像的旋转范围是，随机均匀分布。默认值5，范围[0, 30]
是否进行水平翻转 是否产生水平镜像样本图像
样本区域半径比值 正样本区域的半径比值，感受野中心落在该圆形区域的视为正例样本。
正样本缩放尺度上限 默认值是1.15，有效范围[1.0,4.0]
正样本缩放尺度下限 默认值是0.87，有效范围[0.25,1.0]

c)忽略样本配置

参数名称 参数解释和范围
忽略区域半径比值 感受野在该环形区域的视为忽略样本
忽略区域缩放尺度上限 默认值是2.5，有效范围[1.0,4.0]
忽略区域缩放尺度下限 默认值是0.4，有效范围[0.25,1.0]
忽略难辨认样本 忽略难辨认样本，默认yes
忽略严重遮挡的样本 忽略被严重遮挡的样本，默认yes。
忽略遮挡样本 忽略遮挡，默认no
忽略过小区域数据 忽略过小区域数据，默认值100。
忽略过小高度数据 忽略过小高度数据，默认值10。

d)回归区域样本配置

参数名称 参数解释和范围
坐标回归区域半径比值 感受野中心在该区域的视为回归样本
回归样本缩放尺度上限 默认值是2.5，有效范围[1.0,4.0]
回归样本缩放尺度下限 默认值是0.4，有效范围[0.25,1.0]

e)CNN后验证样本配置

参数名称 参数解释和范围
规范标准长度的方法 默认height，目前只有一个选项
网络输入宽度 网络输入宽度
网络输入高度 网络输入高度
网络输出宽度 网络输出宽度
网络输出高度 网络输出高度
标准长度 CNN网络中输入样本在原始图像中的标准长度

f)校验列表

  • 正样本配置
参数名称 参数解释和范围
感受野宽度 默认值40，有效范围 [24, 72]
感受野高度 默认值40，有效范围 [24, 64]
样本图像宽度 默认384，有效范围[128, 512]
样本图像高度 默认256，有效范围[128, 512]
  • 正样本扰动
参数名称 参数解释和范围
坐标回归区域是否为椭圆 选True，样本最大旋转角度小于等于4
样本最大旋转角度 默认5，有效范围[0, 30]
样本区域半径比值 正例样本区域的半径比值（相对于norm_length），默认值0.15，有效范围[0.1,0.4]
尺度扰动范围 [0.7,1]
  • 忽略样本配置
参数名称 参数解释和范围
忽略区域半径比值 默认0.4，有效范围[0.2，0.5]
尺度扰动范围 0.4~2

注：忽略区域半径比值必须大于样本区域半径比值

3. CNN后验证样本配置

参数名称 参数解释和范围
规范标准长度的方法 默认height，目前只有一个选项
输入宽度 默认值为88
输入高度 默认值为88
输出宽度 输入宽度/16，向上取整；默认值为5
输出高度 输入高度/16，向上取整；默认值为5
标准长度 默认值为50

4. 样本可视化

用户点击 “查看样本示例” 即可显示示例；
可视化分为两个stage: Region Proposal Stage和CNN后验证Stage：
Region Proposal Stage包括示例、正样本示例、负样本示例和忽略样本示例；
CNN后验证示例只包括示例，显示5*5=20张图
